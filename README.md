# Скрипт для получения системной информации с Android

Скрипт `device.py` предназначен для получения детальной информации о подключенном Android-устройстве через ADB (Android Debug Bridge). Он позволяет собрать различные данные о системе устройства, а также вывести информацию о установленных приложениях.

## Возможности

С помощью скрипта можно:
- Получить общую информацию о системе Android-устройства:
  - Модель устройства
  - Версия операционной системы (Android)
  - Номер сборки
  - Процессор
  - Информация о памяти
  - Статус заряда батареи
  - Сетевые параметры
- Получить список установленных приложений на устройстве.
- Экспортировать полученную информацию в формате JSON для дальнейшего анализа.
- Включить логирование всех действий в файл `device.log`.

## Установка

1. Убедитесь, что у вас установлен Python версии 3.7 или выше.
2. Установите ADB (Android Debug Bridge), если он еще не установлен. Для этого можно скачать и установить [Android SDK Platform Tools](https://developer.android.com/studio).
3. Клонируйте репозиторий:

```bash
git clone https://github.com/Android-RU/Android-Device-Info.git
```

4. Перейдите в каталог репозитория:

```bash
cd Android-Device-Info
```

5. Убедитесь, что на вашем устройстве включена отладка по USB (или по Wi-Fi) и что оно подключено к компьютеру.

## Использование

### Основной синтаксис:

```bash
python device.py [OPTIONS]
```

### Опции:

* `--system` — Получить информацию о системе устройства.
* `--apps` — Получить список установленных приложений.
* `--battery` — Получить информацию о заряде батареи.
* `--json` — Экспортировать собранную информацию в файл JSON.
* `--log` — Включить логирование в файл `device.log`.

### Пример использования:

1. Получить информацию о системе и приложениях, экспортируя данные в JSON:

```bash
python device.py --system --apps --json
```

2. Получить только информацию о системе:

```bash
python device.py --system
```

3. Получить информацию о заряде батареи:

```bash
python device.py --battery
```

4. Включить логирование всех операций:

```bash
python device.py --log --system
```
